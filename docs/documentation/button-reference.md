# Button Reference

!!! warning "This Documentation Sucks"
    Cannoli is in early preview. Expect the docs to get better as things progress and get finalized.

    I'd rather just get feedback than commit to making good docs against a moving target.

Cannoli supports both **Xbox** and **Nintendo** button layouts. Pick one in settings.

In **Xbox** layout, A confirms and B goes back. In **Nintendo** layout, it's the opposite.

!!! important
    The tables below use the Xbox layout as the default.

## Platform List

| Button          | Action                              |
|-----------------|-------------------------------------|
| D-Pad Up / Down | Move selection                      |
| A               | Open platform or collection         |
| B               | Go back                             |
| X               | Open settings                       |
| Y               | Open Nonna's Kitchen (when running) |
| Start           | Open context menu                   |
| Select          | Toggle reorder mode                 |

### Reorder Mode

| Button          | Action             |
|-----------------|--------------------|
| D-Pad Up / Down | Move selected item |
| Start / Select  | Confirm reorder    |



## Game List

| Button          | Action                                    |
|-----------------|-------------------------------------------|
| D-Pad Up / Down | Move selection                            |
| A               | Launch game or open folder                |
| B               | Go back                                   |
| X               | Resume game (if save state exists)        |
| L1              | Previous platform                         |
| R1              | Next platform                             |
| Start           | Open context menu                         |
| Select          | Toggle multi-select                       |

### Multi-Select Mode

| Button          | Action               |
|-----------------|----------------------|
| D-Pad Up / Down | Move selection       |
| A               | Toggle item checkbox |
| Start           | Confirm selection    |
| Select          | Toggle reorder mode  |

## Keyboard

| Button  | Action                          |
|---------|---------------------------------|
| D-Pad   | Move keyboard cursor            |
| A       | Input character                 |
| B       | Backspace                       |
| X       | Insert space                    |
| Y       | Cancel input                    |
| L1 / R1 | Move text cursor left / right   |
| L2 / R2 | Move text cursor to start / end |
| Start   | Confirm input                   |
| Select  | Toggle caps                     |

## Color Picker

| Button | Action             |
|--------|--------------------|
| D-Pad  | Move in color grid |
| A      | Select color       |
| B      | Cancel             |
| X      | Open hex input     |
